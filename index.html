@saibotsivad/glopen-routes

These are the available routes that you can import with glopen, as Swagger Editor demo pages so you can explore them more easily.

Note: the SwaggerUI does not yet support OpenAPI 3.1.0, so if you view the docs with that you'll need to manually tweak the top-level openapi property to be openapi: 3.0.0. Some errors will show, since the versions are not forward-compatible.
